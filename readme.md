## 🧠 Project Overview

This project reimagines hiring by removing the need for PDF resumes and introducing:

- 🤖 AI-assisted profile creation  
- 📊 Structured candidate data  
- ⚡ Faster and smarter hiring decisions  

Instead of uploading resumes, candidates interact with AI to build their profiles, making it easier for recruiters to evaluate and compare applicants.

---

## ✨ Key Features

### 👨‍💻 Candidate Side
- AI-powered profile builder
- Add skills, experience, and projects
- Auto-generated structured data
- Profile preview before submission
- Progress tracking & auto-save

### 🧑‍💼 Recruiter Side
- View candidate profiles
- Compare candidates easily
- Shortlist candidates
- Clean dashboard UI

### 🤖 AI Integration
Powered by Groq AI:
- Converts raw input → structured data
- Suggests relevant skills
- Generates summaries
- Enhances candidate profiles
